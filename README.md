# patient-record

A meteor exercise.

## Creating an app

In the terminal
meteor create patient-record

This creates
patient-record.js       # JS for client and server
patient-record.html     # View templates
patient-record.css      # Styles
.meteor                 # internal Meteor files

## Running the app

In the terminal 

cd patient-record
meteor

In the browser
http://localhost:3000
